# Tutorials

HET has two tutorials, one for each module. Neither requires knowledge of the other, so feel free to start wherever you like.

- [Requests tutorial](tutorial-requests.md): progressive enhancement for links, forms, navigation, partial updates, request coordination, and request lifecycle hooks.
- [Components tutorial](tutorial-components.md): component roots, refs, signals, bindings, structural templates, lifecycle, and DOM-to-signal synchronization.
